# Deployment Instructions

To deploy this project to Vercel:

1.  **Push to GitHub**: ensure your latest changes (especially `index.html`!) are committed and pushed to a GitHub repository.
2.  **Import to Vercel**:
    *   Go to [vercel.com](https://vercel.com)
    *   Click "Add New..." -> "Project"
    *   Import your GitHub repository
    *   Vercel will detect it as a static site.
3.  **Deploy**: Click "Deploy".

## Local Development

To run this locally, simply open `index.html` in your browser. Alternatively, you can use a local server like `serve`:

```bash
npx serve .
```
